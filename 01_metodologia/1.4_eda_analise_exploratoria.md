# 1.4 Etapa 3: Análise Exploratória de Dados (EDA)

## Visão Geral

A Análise Exploratória de Dados (EDA) é essencial para compreender padrões, identificar problemas e gerar insights antes da modelagem. Esta etapa produz visualizações e estatísticas descritivas do dataset.

## Objetivos

1. Compreender distribuições de variáveis
2. Identificar correlações entre features
3. Detectar outliers e anomalias
4. Analisar padrões de sobrevivência
5. Gerar visualizações informativas

## Arquivo de Implementação

**Notebook:** `08_codigo/notebooks/EDA_dados_unidos.ipynb`

**Visualizações:** `10_visualizacoes/eda_plots/`

## Análises Realizadas

### 1. Estatísticas Descritivas

- Contagem, média, desvio padrão, min, max
- Distribuição por porte, setor, município
- Taxa de sobrevivência por categoria

### 2. Visualizações

- Histogramas de idade e tempo
- Boxplots por porte
- Heatmaps de correlação
- Gráficos de barras por situação cadastral
- Análise temporal de aberturas e fechamentos

### 3. Análise de Valores Ausentes

- Identificação de features com missing values
- Padrões de ausência (MAR vs MNAR)
- Impacto potencial na modelagem

## Insights Principais

- Empresas mais antigas têm maior taxa de sobrevivência
- Porte está correlacionado com resiliência
- Distribuição geográfica desigual
- Taxa de sobrevivência: ~35.76% (pandemia)
